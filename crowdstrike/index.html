培训大纲

三日课程安排

标准版共 3 天每天 6 小时，也可压缩为 2 天强化版。

1.1大型全球企业安全运维场景导入：资产可见性、终端防护、告警响应、合规报告
1.2Falcon 在企业安全架构中的定位：EPP / EDR / XDR / MDR 的区别
1.3Falcon Console 控制台导览：Dashboard、Host Management、Detections、Investigate
1.4Falcon Sensor 原理与部署：Windows / Linux / macOS，CID，云端通信机制
1.5Prevention Policy 防护策略：分组设计、机器学习检测、IOA 行为阻断、勒索防护
1.6Sensor Update Policy：N / N-1 / N-2 版本策略，分阶段更新，更新风险控制
2.1红蓝对抗基础：红队 / 蓝队 / 紫队，从"看告警"到"看攻击过程"的思维转变
2.2Cyber Kill Chain 七阶段模型：侦察 → 投递 → 利用 → 安装 → C2 → 目标达成
2.3MITRE ATT&CK：Tactics / Techniques / Procedures，检测规则设计，复盘报告
2.4Falcon 检测逻辑：IOC 与 IOA 的本质区别，行为检测的核心价值
2.5场景：钓鱼邮件与恶意文档执行分析
2.6场景：PowerShell 与脚本攻击 / 无文件攻击检测
2.7场景：C2 外联与 Beacon 行为识别
2.8场景：LSASS 凭证访问与 Mimikatz 检测
2.9场景：横向移动检测（PsExec / WMI / RDP / SMB）
2.10场景：勒索软件行为识别与影响范围评估
3.1Falcon 告警分析方法论：Process Tree / Timeline / Network / User 四视角
3.2标准告警分析 12 步骤：从确认基本信息到编写事件记录
3.3告警分级标准：Critical / High / Medium / Low / Informational 响应要求
3.4威胁狩猎 Threat Hunting：假设驱动型、IOC / IOA / ATT&CK-driven Hunting
3.5企业多产品联动：Email Security → EDR → Firewall / Proxy → SIEM → SOAR → ITSM
3.6外包服务响应 SOP：检测 → 分析 → 遏制 → 清除 → 恢复 → 复盘
3.7事件升级触发条件与升级路径：L1 外包 → L2 安全 → 客户本地 → Global SOC
攻防场景

六大实战场景，贯穿完整攻击链

每个场景包含红队行为描述、ATT&CK 映射、Falcon 分析重点与动手实验任务。

SCENARIO 01
钓鱼邮件与恶意文档执行
Initial Access Execution
  • Office / 浏览器启动异常子进程
  • PowerShell / cmd / wscript 执行
  • Parent Process 溯源分析
  • 编码命令 / 隐藏窗口识别
SCENARIO 02
PowerShell 脚本攻击检测
Execution Defense Evasion
  • EncodedCommand / Bypass Policy
  • DownloadString / Invoke-Expression
  • 无文件攻击 Fileless Attack
  • 计划任务 / 注册表持久化
SCENARIO 03
C2 外联与 Beacon 行为
Command & Control
  • 非浏览器进程异常外联
  • HTTPS / DNS / Cloud 伪装
  • Beacon 周期性连接特征
  • 联动 NGFW / Proxy / DNS 安全
SCENARIO 04
凭证访问与 LSASS 防护
Credential Access Privilege Escalation
  • Mimikatz / ProcDump 检测
  • LSASS 访问进程溯源
  • 内存转储文件识别
  • 凭证横向移动关联分析
SCENARIO 05
横向移动检测分析
Lateral Movement Discovery
  • PsExec / WMI / RDP / SMB
  • 同账号多主机短时访问
  • 远程服务创建检测
  • 联动 AD / VPN / 堡垒机日志
SCENARIO 06
勒索软件行为识别
Impact Exfiltration
  • 文件批量修改 / 扩展名变化
  • vssadmin / bcdedit 命令
  • 单点扩散至文件服务器
  • 数据外传迹象识别
